Установка и deploy python на linux mac windows

Установка python

Windows

Установить с официального сайта. В инсталяторе выбрать опцию «Add Python 3.8 to the PATH» для взаимодействия с командной строкой.

Mac

Установка с официального сайта.

Или при помощи Homebrew установка python

Linux Ubuntu

Проверка версии python 

python3 --version

Установка (python3.x для конкретной версии ) 

sudo apt-get update && sudo apt-get install python3

Установка pip

sudo apt install -y python3-pip

Установка venv

sudo apt install -y build-essential libssl-dev libffi-dev python3-dev
sudo apt install -y python3-venv

Протестировать работу окружения (создать директорию проекта)

mkdir testvenv
cd testvenv
source env/bin/activate

(env) user:~/testvenv$ - успешно
