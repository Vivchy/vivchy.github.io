Request

Описание

Пакет requests для взаимодействия с веб-ресурсами посредством HTTP запросов. Он позволяет выполнять запросы типа GET, POST, PUT, DELETE и другие, а также управлять заголовками, параметрами запроса, данными форм и многим другим.

Код 

import requests 

headers = {
    "Accept": "*/*",
    "User-Agent": "Mozilla/5.0 (iPad; CPU OS 11_0 like Mac OS X) 
    AppleWebKit/604.1.34 (KHTML, like Gecko) Version/11.0 Mobile/15A5341f Safari/604.1"
}

url = "https://example.ru"

req = requests.get(url, headers=headers)
src = req.text

headers повышает шанс проверки сервера на робота парсера и обхода блокировки.

Обычно испльзуется в связке с BeautifulSoup и selenium
